# Getting Started

This guide will help you get started with Retropak.

## Prerequisites

Before you begin, ensure you have the following installed:

- Node.js (version 14 or higher)
- npm or yarn package manager

## Installation

To install Retropak, run the following command:

```bash
npm install retropak
```

Or using yarn:

```bash
yarn add retropak
```

## Basic Usage

Here's a simple example to get you started:

```javascript
// Import Retropak
const retropak = require('retropak');

// Your code here
```
