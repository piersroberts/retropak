# @retropak/schema

JSON Schema definition for Retropak manifests.

## Installation

```bash
npm install @retropak/schema
```

## Usage

```javascript
import schema from '@retropak/schema/schema' assert { type: 'json' };
import locales from '@retropak/schema/locales/en' assert { type: 'json' };
```
